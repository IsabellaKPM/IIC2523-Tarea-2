*Stop;Nodo_2;Hola Mundo;Nodo_Kaguya

Start;Nodo_2;Hola Mundo;Nodo_Kaguya
# Simulación 1: se levantan todos y como ya hay 1 nodo proponente, el resto es aceptante.
# Simulación 2: se activan nodos ya activos... no pasa nada.

*Stop;Hola Mundo;Nodo_Kaguya;Nodo_4444
# Nodo_4444 no existe. Se ignora el evento y aunque sea "*"
# no se realizan las bifurcaciones.

Prepare;Nodo_Kaguya;44
# No es nodo proponente en ambas simulaciones, se ignora eventos

Prepare;Nodo_Kaguya!!!;1231

Prepare;Nodo chico :c;1
Accept;Nodo chico :c;1;SET-mejor_pelicula-Kaguya 
# Simulación 1: Funciona el Prepare y Accept
# Simulación 2: No funciona porque "Nodo chico :c" no es proponente

# Simulación 1: existe SET-mejor_pelicula-Kaguya 
# Simulación 2: vacío

Log;mejor_pelicula # Saldrá ["Kaguya"]

Stop;Nodo chico :c
# Simulación 1: Se cae el proponente. Nodo_2 pasa a ser proponente.

# Pero la base de datos no. Una está vacía y la otra tiene SET-mejor_pelicula-Kaguya

Prepare;Nodo_2;3
Accept;Nodo_2;3;DEL-mejor_pelicula
